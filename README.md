# @guriafs/canvas-annotate

> Set of React components to annotate images with shapes on canvas

[![NPM][npm-image]][npm-url]

## Install

```bash
npm install --save @guriafs/canvas-annotate
```

## Usage

```jsx
import React, { Component } from 'react'
```
